# 🪐 ربات و دستیار آپلودر زحل (Zohal TG Uploader)

ربات هوشمند و خط فرمان فوق‌العاده سریع زحل برای انتقال دوطرفه و مستقیم فایل‌ها بین **تلگرام** و **فضای ابری S3** (مانند کلودفلر R2، آروان کلاد، فایلس و مینیو) بدون اشغال حتی ۱ بایت از هارد دیسک سرور شما. 

این ابزار با استفاده از استریم آسنکرون (Async Streams) فایل‌ها را به صورت مستقیم بین تلگرام و فضای ابری لوله‌کشی (Pipe) می‌کند.

---

## ⚡ ویژگی‌های کلیدی زحل

- **انتقال بدون نیاز به هارد دیسک:** فایل‌ها مستقیماً در حافظه موقت (RAM) استریم شده و بدون ذخیره شدن روی سرور شما، منتقل می‌شوند.
- **سرعت آپلود باورنکردنی:** بهینه‌سازی همزمانی (Multipart parallel upload) با پچ اختصاصی پایروگرام با استفاده از ۱۶ کانال موازی.
- **مدیریت کامل از ترمینال (CLI):** ابزار قدرتمند `zohal-up` برای مدیریت سرویس، لاگ‌ها، کاربران مجاز و پیکربندی بدون نیاز به محیط وب.
- **لیست و مدیریت فایل‌های S3:** گشت‌وگذار در پوشه‌های S3، دانلود مستقیم فایل‌ها به تلگرام، تغییر نام و حذف فایل‌ها از داخل چت ربات.
- **ساخت لینک دانلود موقت (Pre-signed URL):** تولید لینک‌های امن و دارای تاریخ انقضا برای فایل‌های S3 از درون تلگرام.
- **پشتیبانی کامل از پروکسی:** تعریف پروکسی Socks5/HTTP اختصاصی برای دور زدن فیلترینگ تلگرام در سرورهای ایران.
- **امنیت بالا و دسترسی محدود:** امکان تعریف لیست سفید کاربران مجاز توسط مدیر اصلی ربات.

## 🚀 راهنمای نصب سریع و آسان (مخصوص سرور اوبونتو و دبیان)

کافیست دستور زیر را کپی کرده و در ترمینال سرور خود اجرا کنید تا پروژه به صورت خودکار نصب و سرویس سیستمی آن فعال شود:

```bash
git clone https://github.com/arsam/ZOHAL-TG-UPLOADER.git && cd ZOHAL-TG-UPLOADER && chmod +x scripts/install.sh && sudo ./scripts/install.sh
```

---

## ⚙️ پیکربندی و راه‌اندازی اولیه

پس از پایان نصب، ابزار مدیریت زحل به دستورات سیستم شما اضافه شده است. برای انجام تنظیمات اولیه:

۱. دستور زیر را در ترمینال وارد کنید:
   ```bash
   zohal-up
   ```
۲. گزینه **7** (`Run Setup Wizard`) را انتخاب کنید.
۳. اطلاعات مورد نیاز شامل API ID، API Hash و توکن ربات تلگرام، به همراه مشخصات فضای ابری S3 خود را وارد کنید.
۴. در پایان، ابزار به صورت خودکار اتصال به تلگرام و S3 را تست کرده و در صورت صحت اطلاعات، ربات را فعال می‌کند.

---

## 🛠️ دستورات مدیریت سرویس

با ابزار `zohal-up` می‌توانید تمام جنبه‌های ربات را مدیریت کنید:

- **مشاهده وضعیت سرویس:** `systemctl status zohal`
- **مشاهده لاگ‌های زنده ربات:** `journalctl -u zohal -f -n 100`
- **راه‌اندازی مجدد ربات:** `systemctl restart zohal`
- **توقف ربات:** `systemctl stop zohal`

---

## 📁 ساختار پوشه‌های پروژه

- `main.py`: هسته اصلی راه‌اندازی ربات به صورت آسنکرون.
- `cli.py`: ابزار خط فرمان هوشمند ربات (`zohal-up`).
- `bot/`: کدهای مربوط به ربات تلگرام (هندلرها، کیبوردها و رفتارهای ربات).
- `core/`: بخش‌های کلیدی شامل دانلودر استریم، کلاینت S3، مدیریت پروکسی و پچ افزایش سرعت آپلود.
- `database/`: دیتابیس لوکال SQLite جهت ذخیره کاربران مجاز، پروکسی‌ها و تاریخچه انتقال‌ها.
